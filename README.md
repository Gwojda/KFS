Hi ! We are 2 students working on kernel from scratch (**KFS**) series of projects at **Ecole 42** !

## What you need

You have to install kvm, grub2, make, gcc, nasm and ld on your host machine. 

## Run our kernel

`git clone https://github.com/Gwojda/KFS.git`
`cd KFS`
`make install`
